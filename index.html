Technologies

Techologies that a Front-End Develoer should master are listed below (Try Mastering them in the order):

  1. Birds-eye view understanding of how internet works, Browsers work, HTTP, DNS, Hosting, URL, Domains, Server/hosting
  2. HTML
  3. CSS
  4. JavaScript
  5. DOM
  6. User-Interface/Interaction/Experience/Usability Design
  7. CLI-Command Line
  8. Practice of Software Engineering - (App Design/Architecture, templates, Git, testing, monitoring, automating, code quality, development methodologies)
  9. Opinionated/customized Toolbox (Webpack, React, Mobx)
  10. Node.js
